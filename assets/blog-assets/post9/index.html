Setting Up Python Environment on Ubuntu

Author: Atul Rajput | Date: 13th Sep, 2024

Setting up a Python environment on Ubuntu is crucial for efficient development, as it allows you to isolate your projects and their dependencies. This guide will walk you through the steps to install Python, create, activate, and deactivate a virtual environment.

Step 1: Install Python and the Virtual Environment Package

By default, Ubuntu comes with Python pre-installed. You can verify this by running:

python3 --version

If Python 3 is not installed, or if you want to update to the latest version, you can install it using the following command:

sudo apt update sudo apt install python3

Next, install the venv package, which allows you to create virtual environments:

sudo apt install python3-venv

Step 2: Create a Virtual Environment

Once Python and venv are installed, you can create a new virtual environment for your project. This will allow you to manage dependencies specific to your project without affecting your system-wide Python installation.

To create a virtual environment, run:

python3 -m venv myenv

Here, myenv is the name of your virtual environment. You can replace it with any name you prefer.

Step 3: Activate the Virtual Environment

After creating the virtual environment, you need to activate it to start using it. This ensures that all the Python packages you install are stored within this environment.

To activate the virtual environment, use the following command:

source myenv/bin/activate

Once activated, your terminal prompt will change to indicate that you are now working inside the virtual environment.

Conclusion

Using virtual environments is a best practice in Python development. It allows you to manage dependencies in isolation, ensuring that your projects remain independent and free from version conflicts. Following these steps, you can easily set up, activate, and manage Python virtual environments on Ubuntu.
